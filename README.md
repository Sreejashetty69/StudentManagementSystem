# Student Management System (Java)

This is a beginner Java project created to manage student records using Object-Oriented Programming and file handling.

## Features
- Add student details
- View all students
- Search student by ID
- Update student information
- Delete student record
- Data is stored in a file (persistent storage)

## Technologies Used
- Java
- OOP Concepts (Classes and Objects)
- File Handling

## What I Learned
- Creating and using classes in Java
- Implementing CRUD operations
- Reading and writing data to files
- Building menu-driven console applications
- Using GitHub to upload and manage projects
